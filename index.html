What is a div?

  • The div tag defines a division or a section in an HTML document.
  • It is also used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

  • Helps separate the the bar from other elements of the page, making it easier to identify and prevents confusion

When it is useful:

  • When you want to divide parts/sections of an HTML document.
  • Allows to style a whole section without applying classes individually to each element.
